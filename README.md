# ~/.dotfiles

## Instructions

### Run Migration On Old Computer Before Setup

```sh
bash -c "$(curl -LsS https://raw.github.com/wking-io/dotfiles/master/src/os/migrate.sh)"
```

### Run Setup

```sh
bash -c "$(curl -LsS https://raw.github.com/wking-io/dotfiles/master/src/os/setup.sh)"
```
